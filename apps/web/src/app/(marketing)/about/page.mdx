# Bridging Ancient Tradition with Modern Astronomy

Hilal Vision is a precision astronomical web platform dedicated to predicting and visualising Islamic crescent moon (هلال) sightings worldwide. It is built for the 1.8 billion Muslims who rely on the lunar calendar for religious observances, and for the astronomers and scholars who study it.

## Our Mission

The Islamic lunar calendar is one of humanity's oldest scientific traditions - the beginning of each sacred month has been determined by the physical sighting of the new crescent moon for over 1,400 years. Yet in the modern world, this practice is fragmented: different countries announce the start of Ramadan on different days, and communities lack transparent, data-driven tools to understand why.

Hilal Vision exists to answer one question with the precision it deserves: **"Will the new crescent moon be visible tonight from my location - and why?"**

We implement the internationally recognised **Yallop (1997)** and **Odeh (2004)** visibility criteria - the same standards used by Islamic calendar authorities in the UK, Malaysia, and the international astronomical community - and present the results with the visual clarity and interactivity that a 21st-century audience expects.

This platform is not a religious authority. It is a scientific instrument. It presents mathematical predictions, historical data, and comparative calendrical analysis side-by-side, empowering individuals and communities to engage critically and transparently with Islamic timekeeping.

## Who Is It For?

### Muslim Communities
Get a clear, location-specific prediction for crescent visibility before Ramadan, Eid, and every new Hijri month - without jargon.

### Astronomers & Researchers
Access raw q-values, ARCV/DAZ parameters, Odeh V-values, and the full ICOP historical dataset for validation and research.

### Islamic Calendar Scholars
Compare Astronomical, Umm al-Qura, and Tabular Hijri calendars side-by-side. Understand exactly where and why civic calendars diverge from physical astronomy.

## Platform Tools

- **3D Globe & Visibility Map**: Interactive WebGL globe and 2D Leaflet map showing crescent visibility zones worldwide using Yallop q-values. Includes real-time cloud cover overlay and Best-Time-to-Observe calculator.
- **Moon Phase Dashboard**: Current lunar phase, illumination, age, Sun & Moon altitude chart, 30-day phase calendar strip, and Yallop/Danjon methodology charts.
- **Hijri Calendar**: Triple-engine calendar supporting Astronomical (astronomy-engine), Umm al-Qura (KACST), and Tabular algorithms. Includes a 'Compare to Heavens' divergence overlay.
- **Horizon View**: Local horizon simulator showing the crescent moon's position relative to the setting sun, with ARCV and DAZ annotations.
- **ICOP Archive**: 1,000+ authentic historical crescent sighting records from the Islamic Crescents' Observation Project (ICOP), spanning 1438–1465 AH.

## Technology

Built with modern web technologies for performance, accuracy, and global reach:

- **React 15 + Next.js App Router**: Frontend framework and build tool
- **Mapbox + Globe.gl**: 2D map and 3D WebGL globe
- **astronomy-engine**: VSOP87/ELP2000 sun & moon position algorithms (Don Cross)
- **tRPC + Drizzle ORM + PostgreSQL**: Type-safe API and database layer
- **Clerk + Vercel KV**: Authentication, edge caching, and rate limiting
- **Capacitor.js**: Native iOS & Android packaging

## Data Sources & Attributions

Hilal Vision stands on the shoulders of decades of peer-reviewed astronomical research.

- **Yallop (1997) Criterion**: B.D. Yallop, HM Nautical Almanac Office - foundational q-value crescent visibility formula.
- **Odeh (2004) Criterion**: Mohammad Odeh - V-value refinement trained on 737 ICOP sighting observations.
- **Islamic Crescents' Observation Project (ICOP)**: International Astronomical Center - over 1,000 historical crescent sighting records used in the Archive.
- **astronomy-engine by Don Cross**: High-precision VSOP87/ELP2000 planetary and lunar position library for JavaScript.
- **Umm al-Qura Calendar (@umalqura/core)**: KACST pre-computed tables for the official Saudi Arabian civic Hijri calendar.
- **Open-Meteo**: Free, open-source weather API providing real-time cloud cover data.
